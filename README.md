# Grill Park Restaurant Website

Static React application for Grill Park (Tiruppur) with Tailwind CSS styling.

## Features

- Functional components for layout
- Dark + orange theme with responsive design
- Navbar, Hero, About, Services, MenuHighlights, Reviews, Contact, Footer
- Call now & Order online buttons
- Google Maps embed on contact section

## Setup

```bash
npm install
npm start
```

Navigate to http://localhost:3000 to view the site.

Styles are managed with Tailwind; configuration files include `tailwind.config.js` and `postcss.config.js`.

Feel free to modify the components in `src/components` and update content as needed.
